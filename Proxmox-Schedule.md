# Proxmox Schedule

หากต้องการตรวจสอบหรือดูตารางเวลาที่ตั้งไว้ใน Proxmox (เช่น การสำรองข้อมูล หรือการทำ Replication) ให้ไปที่เมนู Datacenter ในหน้า Web GUI จากนั้นคลิก Backup หรือ Replication เพื่อดูงานที่ตั้งเวลาไว้ โดย Proxmox จะใช้รูปแบบ Calendar Event ของ systemd ในการกำหนดเวลา
## วิธีการตรวจสอบด้วยเครื่องมือจำลองตารางเวลา (Schedule Simulator)
Proxmox มีฟีเจอร์จำลองตารางเวลาเพื่อความถูกต้อง

   1. ไปที่ Datacenter > Backup
   2. เลือกหรือสร้าง Backup Job (งานสำรองข้อมูล)
   3. คลิกปุ่ม Schedule Simulator ที่มุมขวาบน ระบบจะแสดงรอบเวลาและเวลาถัดไปที่งานจะเริ่มทำงานอย่างแม่นยำ

## วิธีการตรวจสอบตารางเวลาผ่าน Command Line (CLI)
หากคุณเขียนคำสั่งเวลาเอง สามารถนำไปทดสอบบนระบบ Proxmox CLI ได้โดยตรงด้วยคำสั่งนี้ (เปลี่ยน 'YOUR_SCHEDULE' เป็นรูปแบบที่คุณต้องการ):
systemd-analyze calendar 'YOUR_SCHEDULE'
## ตัวอย่างรูปแบบตารางเวลาที่ใช้บ่อยใน Proxmox

* */30: ทำงานทุกๆ 30 นาที
* 00:30: ทำงานทุกวัน เวลา 00:30 น.
* mon..fri 22:00: ทำงานวันจันทร์ถึงวันศุกร์ เวลา 22:00 น.
* sat *-1..7 15:00: ทำงานวันเสาร์แรกของทุกเดือน เวลา 15:00 น.

คุณต้องการให้แนะนำวิธีเขียน ตารางเวลาแบบเฉพาะเจาะจง (เช่น สำรองข้อมูลวันเว้นวัน หรือระบุวันที่ต้องการ) หรือต้องการให้ช่วย ตรวจสอบไฟล์ Cron สำหรับสคริปต์ที่ตั้งให้รันตอนเปิดเครื่อง (VM Startup) ดี?
